## 🎯 Projects

### 1. NFL Game Outcome Analysis (2026)
**Collaborative Classification Modeling | My Role: Decision Tree Implementation**

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![SAS](https://img.shields.io/badge/SAS-9.4+-1f425f.svg)
![Team Project](https://img.shields.io/badge/Team-4%20Members-orange.svg)

**Research Question**: Which offensive and defensive performance metrics have the greatest impact on winning in the NFL?

**My Contribution**:
- 🌳 Developed and optimized Decision Tree classification models using Python (scikit-learn)
- 📊 Tested multiple data specifications (V1: seasonal averages vs V2: single-game stats)
- 🎯 Achieved 87.03% validation accuracy with optimized V2 80/20 configuration
- 📈 Conducted comprehensive feature importance analysis
- 🔍 Identified `def_total_off_points` as the dominant predictor (51.39% of split importance)

**Team Models Comparison**:
| Model | Accuracy | Team Member | Tool | Key Strength |
|-------|----------|-------------|------|--------------|
| Neural Network | 92.66% | Jalen | SAS | Highest accuracy |
| Logistic Regression | 92.00% | Keila | SAS | Best interpretability |
| Random Forest | 87.61% | Darshil | Python | Robust ensemble |
| **Decision Tree** | **87.03%** | **Fu Wei (Me)** | **Python** | **Highly interpretable** |

**Key Team Findings**:
- 🛡️ Defense is #1 predictor: Opponent points allowed accounts for 27-51% of importance across all models
- 🎯 Red zone execution > Total yardage: Touchdowns outrank yards in predictive power
- 📊 Single-game data dramatically outperforms seasonal averages (87-93% vs 65-80% accuracy)

**Dataset**: 6,972 NFL games (2012-2024) | 175 performance metrics  
**Course**: MIS5560 Data Science | Oakland University | **Team Size**: 4 members
